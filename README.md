# myDockerBuild
A custom and personal docker build system

To deploy :

1.
```bash
$ git clone git@github.com:Fclem/myDockerBuild.git your_folder_name && rm -fr your_folder_name/.git
```

2.
```bash
$ cd your_folder_name && ./init_docker-build-system.sh
```
